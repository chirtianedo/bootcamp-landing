üéØ DESIGN TIPS FOR THRIVE ARCHITECT
=====================================================

1. BACKGROUNDS:
   - Use pure black (#000000) as base
   - Add subtle gradient overlays
   - Use blur effects for depth

2. BUTTONS:
   - Primary: Green gradient (#34d399 to #10b981)
   - Add glow/shadow effects
   - Border radius: 12-16px

3. CARDS/BOXES:
   - Semi-transparent white (5% opacity)
   - Border: 1-2px, primary color at 20% opacity
   - Border radius: 24px
   - Add hover effects (lift, scale)

4. TYPOGRAPHY:
   - Headlines: 48-96px, Bold/Black weight
   - Subheads: 24-32px, Medium/Semibold
   - Body: 16-20px, Regular
   - Use gradient on key phrases

5. SPACING:
   - Section padding: 80-128px top/bottom
   - Element spacing: 32-64px
   - Card grid gaps: 24-32px

6. ANIMATIONS:
   - Fade in on scroll
   - Hover lift on cards (move up 10px)
   - Button hover glow
   - Smooth transitions (300ms)

7. VIDEO EMBED:
   - Aspect ratio: 16:9
   - Max width: 1200px
   - Add play button overlay

8. RESPONSIVE:
   - Stack cards on mobile
   - Reduce font sizes 30-40%
   - Maintain readability
